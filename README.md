# Front-end do Pokemon

Este é um projeto front-end do Pokemon, desenvolvido com React.js, TypeScript e Vite.

## Pré-requisitos

Antes de começar, certifique-se de ter os seguintes requisitos instalados em sua máquina:

- [Node.js](https://nodejs.org/) (v18.18.2)
- [npm](https://www.npmjs.com/) ou [Yarn](https://yarnpkg.com/)

## Scripts Disponíveis

No diretório do projeto, você pode executar os seguintes scripts:

- `npm run dev` ou `yarn dev`: Inicia o servidor de desenvolvimento.
- `npm run build` ou `yarn build`: Compila o aplicativo para produção na pasta `dist`.

## Mais Informações

Para obter mais informações sobre o Vite, consulte a documentação do Vite em [https://vitejs.dev/](https://vitejs.dev/).
